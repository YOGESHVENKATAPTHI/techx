## Getting Started

### Prerequisites

- Node.js v18+  
- npm (v9+) or Yarn (v1.22+)  
- PostgreSQL (or SQLite/MySQL) for database  

### Installation

Clone the repo:
git clone https://github.com/YOGESHVENKATAPTHI/techx.git
cd techx

ensure you have installed node.js and run:
npm install

### Running in Development
npm run dev
